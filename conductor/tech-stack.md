# Technology Stack

## Core Runtime
- **Electron:** For building the cross-platform desktop application shell.

## Frontend Environment
- **React:** The primary UI library for building the "ZenState" interface.
- **Vite:** The build tool and development server, ensuring high-speed HMR and optimized builds.
- **TypeScript:** Providing type safety and enhanced developer experience across the codebase.

## Styling & Animation
- **Tailwind CSS:** For rapid, utility-first styling consistent with the "Organic Minimalism" aesthetic.
- **Framer Motion:** Powering the "Fluid & Ethereal" transitions and reactive UI state changes.
- **Lucide React:** Providing a clean, consistent icon set.

## Data & Storage
- **SQLite (better-sqlite3):** A robust, local database for storing user settings, focus metrics, and Neural Codex content.
- **LocalStorage:** Used for lightweight, non-relational persistence where appropriate.

## Key Utilities
- **clsx & tailwind-merge:** For dynamic and conflict-free CSS class management.
- **React Router:** For seamless navigation between the Dashboard, Neural Codex, and Settings.
